如果这是你第一次拍屏，那我们理解，有人可能确实还不知道群里为什么一直强调“截图而不是拍屏”。现在我来讲一遍原因：

🧠 为什么不推荐拍屏？
- 拍屏画质差 → 日志文字糊得像疙瘩汤
- 拍屏信息不全 → 关键内容缺失
- 拍屏角度奇葩 → 光斑、反光、摩尔纹，还得答者练斜视眼看图
- 拍屏后还经常发虚、拍不全、遮住重点 → 全是信息噪音

🎯 这些都会大幅增加答者的理解成本，甚至根本没法看问题出在哪，所以我们一律要求使用截图工具（Win+Shift+S / QQ / 微信都可以）。
